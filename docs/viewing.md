
# Viewing Mermaid Diagrams in VS Code

## Recommended Extensions
- **Markdown Preview Mermaid Support**
- **Mermaid Editor**

## Previewing `.mmd` files
1. Open the `.mmd` file in VS Code.
2. Use the extensionâ€™s preview command (e.g., `Markdown: Open Preview to the Side`) or the Mermaid preview.
3. For Markdown files, wrap Mermaid code blocks in triple backticks with `mermaid` and use Markdown preview.

## Exporting to Images
- Some extensions allow export to PNG/SVG from the preview.
- Save exported files under `assets/exports/<project>/<diagram>.svg` (optional).
